# 核心规则

DownCollect 是一个选牌计分的卡牌游戏。

## 涉及要素

- 所有卡牌：目前 DownCollect 采用扑克牌牌堆，包含 54 张牌（52 张普通牌 + 2 张不同的 Joker）。普通牌分为四种花色（红桃、黑桃、方块、梅花），每种花色有 13 张牌（A, 2-10, J, Q, K）。
- 区域：游戏中有三个主要区域：玩家手牌区、牌堆区、临时揭示处理区域、卡牌展示区域。
- 玩家：游戏通常由 2-4 名玩家参与。
- 角色：每位玩家在游戏开始时从角色池中随机抽取 2 个角色，玩家可以选择其中一个角色作为本次游戏的角色，角色具有在特殊阶段可以使用的能力。
- 公共计分规则和玩家计分规则：游戏中存在一些公共的计分规则，所有玩家都适用，同时每个玩家可能会有一些特定的计分规则，这可能由玩家抽选的计分规则提供，玩家彼此之间的计分规则不可见。
- 阶段：游戏分为多个阶段，包含起始阶段，揭示阶段，选牌阶段和计分阶段。

## 流程

### 起始阶段

1. 游戏开始前，所有玩家从角色池中随机抽取 2 个角色，玩家可以选择其中一个角色作为本次游戏的角色。
2. 从所有卡牌的集合中移除随机的 5 张卡牌（暂定），这些卡牌在本轮游戏中不参与。
3. 将剩余卡牌随机给每位玩家分发 1 张牌（暂定，未来可能为0，若设计为2或更多则除非通过卡牌效果或角色能力，顺序不可改变），玩家只能看到自己的牌。
4. 牌堆剩余的卡牌洗混放在桌面中央，形成公共牌堆。

### 揭示阶段

1. 每位玩家轮流行动，每位玩家都行动了一次后，进入下一个轮次。（第一轮的起始玩家随机决定，之后每一轮的起始玩家为上一轮的起始玩家的下一位玩家）
    1.1. 玩家从牌堆顶抽取一张牌，仅自己可见。
    1.2. 如果玩家有手牌，可以选择将抽到的牌与一张手牌交换（抽到的牌进入手牌，手牌成为本次需要处理的卡牌），或者选择不交换。其他玩家不会知道是否发生了交换，也不会看到被交换到手中的牌。
    1.3. 将最终确定的卡牌展示给所有玩家，所有玩家都可以看到这张牌。
    1.4. 玩家必须根据该卡牌效果进行相应的操作，具体卡牌效果详见卡牌效果文档（[卡牌效果文档](./card_effects.md)）。
    1.5. 在效果处理完毕后，若卡牌效果未指定该卡牌放置位置，则该卡牌以暗置形式放置在卡牌展示区域的第M行的最左侧未被占用的位置，其中 M 为当前轮次数（第一轮为1，第二轮为2，以此类推）。
2. 当所有玩家都完成了他们的行动后，进入下一个轮次，重复步骤 1， 直到完成 6 轮（暂定）。

特殊地，角色拥有一些在特殊时间节点可以使用的能力，游戏设计中该阶段可以包含如下时间节点：
- 揭示阶段开始时
- 揭示阶段轮次开始时
- 玩家行动开始后
- 玩家抽牌后，交换决策前
- 玩家交换决策后，卡牌展示前
- 玩家揭示卡牌后，卡牌效果处理前
- 卡牌效果处理后并放置后
- 玩家行动结束前
- 揭示阶段轮次结束时
- 揭示阶段结束时

### 选牌阶段

1. 每位玩家轮流行动，每位玩家都行动了一次后，进入下一个轮次。（第一轮的起始玩家为揭示阶段最后一轮的起始玩家的下一位玩家，之后每一轮的起始玩家为上一轮的起始玩家的下一位玩家）
    1.1. 玩家可以选择从卡牌展示区域的任意一行选择一张卡牌加入自己的手牌的最右侧。
    1.2. 玩家选择的卡牌彼此之间不可见，且玩家只能看到自己手牌中的卡牌。
2. 当所有玩家都完成了他们的行动后，进入下一个轮次，重复步骤 1， 直到完成 4 轮（暂定）。

特殊地，角色拥有一些在特殊时间节点可以使用的能力，游戏设计中该阶段可以包含如下时间节点：
- 选牌阶段开始时
- 选牌阶段轮次开始时
- 玩家行动开始后
- 玩家选择卡牌前
- 玩家选择卡牌后
- 玩家行动结束前
- 选牌阶段轮次结束时
- 选牌阶段结束时

### 计分阶段

玩家同时根据自己的手牌进行计分。

- 每张卡牌都具有一个基本分值，卡牌的基础分值为扑克点数，Joker 的基础分值为 0 分，A 的基础分值为 1 分，2-10 的基础分值为其点数，J、Q、K 的基础分值分别为 11、12、13 分。
- 每张卡牌会提供一个特殊计分规则，详见计分规则文档（[计分规则文档](./scoring_rules.md)）。
- 玩家角色可能会提供额外的计分规则，详见角色能力文档（[角色能力文档](./character_abilities.md)）。
- 公共计分规则适用于所有玩家，详见计分规则文档（[计分规则文档](./scoring_rules.md)）。
- 玩家私有计分规则仅适用于特定玩家，玩家之间不可见，详见计分规则文档（[计分规则文档](./scoring_rules.md)）。

计分顺序如下：
1. 玩家首先根据角色计分规则进行计分，如果未明文规定跳过剩下的计分步骤，否则继续进行下一步。
2. 玩家根据玩家私有计分规则进行计分，如果未明文规定跳过剩下的计分步骤，否则继续进行下一步。
3. 玩家根据公共计分规则进行计分，如果未明文规定跳过剩下的计分步骤，否则继续进行下一步。
4. 从玩家手牌中最右侧的卡牌开始，按照从右到左的顺序，玩家依次根据每张卡牌的特殊计分规则进行计分，如果未明文规定跳过剩下的计分步骤，否则继续根据下一张卡牌的特殊计分规则进行计分。当卡牌按照其特殊计分规则进行计分时，其只能看见包含自己和左侧卡牌的部分手牌信息（如果有的话），但无法看见右侧卡牌的信息。

最终分数最少的玩家获胜，如果存在多个玩家分数相同且为最低分，则这些玩家并列获胜。

特殊地，角色拥有一些在特殊时间节点可以使用的能力，游戏设计中该阶段可以包含如下时间节点：
- 计分阶段开始时
- 玩家计分前
- 私有计分规则计分前
- 私有计分规则计分后
- 公共计分规则计分前
- 公共计分规则计分后
- 卡牌特殊计分规则计分前
- 卡牌特殊计分规则计分后
- 玩家计分后
- 计分阶段结束时
